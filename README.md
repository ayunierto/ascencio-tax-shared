# Ascencio Tax - Shared Package

Código compartido entre API, Web y Mobile.

## Contenido

- **Types & Interfaces**: Tipos TypeScript compartidos
- **Schemas**: Validaciones Zod
- **Constants**: Enums y constantes

## Instalación

```bash
npm install github:ayunierto/ascencio-tax-shared#main
```

## Desarrollo

```bash
npm install
npm run build
```
